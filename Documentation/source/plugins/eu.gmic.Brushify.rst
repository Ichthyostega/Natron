G’MIC Brushify node
===================

*This documentation is for version 1.0 of G’MIC Brushify.*

Description
-----------

Wrapper for the G’MIC framework (http://gmic.eu) written by Tobias Fleischer (http://www.reduxfx.com) and Frederic Devernay.
